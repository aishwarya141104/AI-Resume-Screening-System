# AI-Resume-Screening-System

## Overview

AI Resume Screening System is a Java Swing desktop application that automates resume screening and candidate ranking. The system extracts candidate information from PDF resumes, identifies technical skills, calculates scores, and ranks candidates based on their qualifications.

## Features

* Upload PDF Resume
* Extract Candidate Name
* Extract Email Address
* Detect Technical Skills
* Calculate Resume Score
* Rank Candidates Automatically
* Save Report as TXT File
* Modern Gradient GUI

## How to Run

1. Open the project in IntelliJ IDEA.
2. Add Apache PDFBox libraries.
3. Run ResumeScreenUI.java.
4. Upload PDF resumes.
5. View candidate rankings.


## Output

The system generates:

* Candidate Information
* Skill Analysis
* Resume Score
* Candidate Ranking

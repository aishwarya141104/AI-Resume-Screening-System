# AI-Resume-Screening-System

## Overview
AI Resume Screening System is a Java Swing application that automates resume screening and candidate ranking.

## Features
- Upload PDF Resume
- Extract Candidate Name
- Extract Email Address
- Detect Technical Skills
- Calculate Resume Score
- Rank Candidates Automatically
- Save Report as TXT File
- Modern Gradient GUI

## Technologies Used
- Java
- Swing
- Apache PDFBox

## Skills Evaluated
- Java
- Python
- HTML
- CSS
- JavaScript
- Power BI
- Figma
- C++
- MySQL

## How to Run
1. Open project in IntelliJ IDEA.
2. Add Apache PDFBox libraries.
3. Run ResumeScreenUI.java.
4. Upload PDF resumes.
5. View candidate ranking.

## Output
The system generates:
- Candidate Information
- Skill Analysis
- Resume Score
- Candidate Ranking
